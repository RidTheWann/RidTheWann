### About Me

Saya adalah seorang pengembang perangkat lunak yang berfokus pada efisiensi sistem dan performansi. Saya memiliki ketertarikan mendalam pada low-level programming (Emulator dan System Tools) serta teknologi web modern.

Saat ini saya sedang mengembangkan ekosistem **WeaR**, sebuah rangkaian software utilities dan proyek eksperimental.

- Currently working on **WeaR-emu (PlayStation 4 Emulator)** & **WeaR-Defender**.
- Currently learning **Advanced Rust Memory Management** & **Kernel Development**.
- Focus: Combining C++/Rust performance with TypeScript flexibility.
